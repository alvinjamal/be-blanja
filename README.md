# Store.id Backend
# Run Locally

Clone the project

https://github.com/alvinjamal/be-blanja.git

Go to the project directory

  cd be-blanja

Install dependencies

  npm install

Start the server

  npm run dev

Environment Variables

To run this project, you will need to add the following environment variables to your .env file Start the server

MAIL_USERNAME=
MAIL_PASSWORD=
OAUTH_CLIENTID=
OAUTH_CLIENT_SECRET=
OAUTH_REFRESH_TOKEN=

DB_USER=
DB_HOST=
DB_NAME=
DB_PASS=
DB_PORT=
JWT_KEY=
HOST=

PHOTO_NAME=
PHOTO_KEY=
PHOTO_SECRET=
